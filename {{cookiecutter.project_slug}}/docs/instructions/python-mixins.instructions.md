# Mixins Instructions

When implementing mixin classes in this project, follow these guidelines:
1. Make the mixin inherit from the base class it is intended to be mixed with, if applicable. This is a good practice as it serves as documentation and can help with type checking.
1. Add stubs for any expected method or attribute that the mixin relies on.
1. Define the mixin class with a clear and descriptive name that indicates its purpose.
